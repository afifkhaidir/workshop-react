React Event Handler

React event are named using camelCase and pass a function rather than a string

  • HTML

    <button onclick="handleClick()">Order Now</button>

  • React

    <button onClick={this.handleClick}>Order Now</button>

List and Keys

Every component inside the array should have key property. It would help react to identify which items have changed

This list

  const foods = ['Siomay', 'Batagor', 'Rujak'] 

  const listItems = foods.map((food, index) => <li key={index}>{food}</li>)

Will be transformed

  const listItems = (
    <li key=0>Siomay</li>
    <li key=1>Batagor</li>
    <li key=2>Rujak</li>
  )
